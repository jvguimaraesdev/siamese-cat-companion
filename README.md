<h2>About</h2>

<p>
  <strong>EN:</strong> Interactive Siamese cat built with HTML, CSS and JavaScript that reacts to user actions in real time.
</p>

<p>
  <strong>PT:</strong> Gato siamês interativo desenvolvido com HTML, CSS e JavaScript que reage às ações do usuário em tempo real.
</p>

<!-- FUNCIONALIDADES -->

<h2>Features</h2>

<p><strong>EN:</strong>
  <ul>
    <li>Eyes follow the mouse cursor;</li>
    <li>Meow button with visual effect;</li>
    <li>Interactive sounds (meow, purr, and feeding);</li>
    <li>Drag-and-drop feeding system;</li>
    <li>Language switching (EN / PT);</li>
    <li>Smooth animations.</li>
  </ul>
</p>

<p><strong>PT:</strong>
  <ul>
    <li>Olhos seguem o cursor do mouse;</li>
    <li>Botão de miar com efeito visual;</li>
    <li>Sons interativos (miado, ronronar e alimentação);</li>
    <li>Sistema de alimentação com arrastar;</li>
    <li>Alternância de idioma (EN / PT);</li>
    <li>Animações suaves.</li>
  </ul>
</p>

<!-- INTERAÇÕES -->

<h2>Interactions</h2>

<p><strong>EN:</strong>
  <ul>
      <li>Hover the cat to make it purr;</li>
      <li>Click the button to hear a meow;</li>
      <li>Drag the food to feed the cat;</li>
      <li>Watch the eyes follow your cursor.</li>
  </ul>
</p>

<p><strong>PT:</strong>
  <ul>
    <li>Passe o mouse sobre o gato para fazê-lo ronronar;</li>
    <li>Clique no botão para ouvir o miado;</li>
    <li>Arraste a comida até o gato para alimentá-lo;</li>
    <li>Observe os olhos acompanhando o cursor.</li>
  </ul>
</p>
