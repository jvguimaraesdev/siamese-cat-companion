<!-- SOBRE O PROJETO -->

<h2>About</h2>

<p>
  <strong>EN:</strong> Interactive Siamese cat built with HTML, CSS and JavaScript that reacts to user actions in real time.
</p>

<p>
  <strong>PT:</strong> Gato siamês interativo desenvolvido com HTML, CSS e JavaScript que reage às ações do usuário em tempo real.
</p>

<!-- FUNCIONALIDADES -->

<h2>Features</h2>

<p><strong>EN:</strong></p>
  <ul>
    <li>Eyes follow the mouse cursor;</li>
    <li>Meow button with visual effect;</li>
    <li>Interactive sounds (meow, purr and feeding);</li>
    <li>Drag-and-drop feeding system;</li>
    <li>Language switching (EN / PT);</li>
    <li>Smooth animations.</li>
  </ul>

<p><strong>PT:</strong></p>
  <ul>
    <li>Olhos seguem o cursor do mouse;</li>
    <li>Botão de miar com efeito visual;</li>
    <li>Sons interativos (miado, ronronar e alimentação);</li>
    <li>Sistema de alimentação com arrastar;</li>
    <li>Alternância de idioma (EN / PT);</li>
    <li>Animações suaves.</li>
  </ul>

<!-- INTERAÇÕES -->

<h2>Interactions</h2>

<p><strong>EN:</strong></p>
  <ul>
      <li>Move the mouse to make the cat's eyes follow the cursor;</li>
      <li>Hover over the cat to make it purr;</li>
      <li>Click the button to hear a meow;</li>
      <li>Drag the food to feed the cat.</li>
  </ul>

<p><strong>PT:</strong></p>
  <ul>
    <li>Mova o mouse para fazer os olhos do gato seguirem o cursor;</li>
    <li>Passe o mouse sobre o gato para fazê-lo ronronar;</li>
    <li>Clique no botão para ouvir o miado;</li>
    <li>Arraste a comida para alimentar o gato.</li>
  </ul>

<!-- PROPÓSITO -->

<h2>Purpose</h2>

<p>
  <strong>EN:</strong> This project was developed to practice DOM manipulation and event-driven logic using JavaScript. The focus was on handling real-time interactions such as mouse tracking, drag-and-drop mechanics, and synchronizing visual and audio feedback.
</p>

<p> 
  <strong>PT:</strong> Este projeto foi desenvolvido com o objetivo de praticar manipulação do DOM e lógica orientada a eventos com JavaScript. O foco foi trabalhar interações em tempo real como rastreamento do mouse, drag-and-drop e sincronização entre efeitos visuais e sonoros.
</p>
